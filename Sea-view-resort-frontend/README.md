# Frontend - Sea View Resort

Sistema de reservas de hotel con React.

## Instalar

```bash
npm install
```

## Correr

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Features

- Login/registro con JWT
- Ver habitaciones y hacer reservas
- Panel admin para gestionar rooms/users/reservas
- Perfil con historial de reservas

## Tech

- React + Vite
- TailwindCSS
- React Router
- Axios
- Context API
